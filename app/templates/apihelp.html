LunchRoll API Documentation

Introduction

Provides a general description of the LunchRoll API.

client

Provides web services for requesting Google queries.

APIDescription
POST client/search

Retrieves restaurants in the area
POST client/suggest

Retrieves restaurants based on a user or group's preferences

user

Provides the endpoints for user related queries

APIDescription
POST user/updateUser

Creates a new user or modifies an existing one
POST user/getUser

Gets user's data
POST user/getGroups

Gets user's groups
POST user/getFriends

Retrieves all of user's friends
POST user/addFriend

Adds a friend
POST user/removeFriend

Removes a friend

group

Provides the endpoints for user related queries

APIDescription
POST group/updateGroup

Creates a new group or modifies an existing one
POST group/getGroup

Gets group's data
POST group/deleteGroup

Deletes a group
