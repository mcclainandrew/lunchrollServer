LunchRoll API Documentation

Introduction

Provides a general description of the LunchRoll API.

client

Provides web services for requesting Google queries.

APIDescription
POST client/nearbyAny

Retrieves any restaurant in the area of the user
POST client/nearbySpecific

Retrieves restaurants based on the type of food passed in
POST client/nearbySuggested

Retrieves restaurants based on the user's previous choices

user

Provides the endpoints for user related queries

APIDescription
POST user/updateUser

Creates a new user or modifies an existing one
