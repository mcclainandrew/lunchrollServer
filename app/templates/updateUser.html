POST user/updateUser

Creates a new user or modifies an existing one.

Request Information

URI Parameters

None.

Body Parameters

User Fields

Name Description Type Additional Information
userId The user's ID String Required
If the user does not have an Id, set to 0 to create a user
username The user's username String Required
The user's username. This may not be updated once set.
password The user's password String Required
The user's new Password
email The user's email String Required
The user's new Email

Request Formats

application/json

{"userId": "0", "username":"lonelyjoe", "password":"soAlone", "email":"partyguy123@plentyoffish.com"}

Response Information

Resource Description

application/json

Response Formats

application/json

{
  "data": {
    "Success": true, 
    "userId": 6
  }
}
