POST user/login

Checks a supplied password and username/email against the user databased.

Request Information

URI Parameters

None.

Body Parameters

User Fields

Name Description Type Additional Information
password The user's password String Required
username The username of the user String Optional
must supply either a username or email
email The email of the user String Optional
must supply either a username or email

Request Formats

application/json

{"userId":"3"}

Resource Description
