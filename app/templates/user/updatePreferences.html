POST user/updatePreferences

Creates a new user preference entry or modifies an existing one.

Request Information

URI Parameters

None.

Body Parameters

User Fields

Name Description Type Additional Information
userId The user's ID String Required
asian The weight for asian food String Required
american The weight for american food String Required
italian The weight for italian food String Required
mexican The weight for mexican food String Required
indian The weight for indian food String Required
greek The weight for greek food String Required

application/json

{"userId":"3", "asian":"1", "american":"1","italian":"1", "mexican":"0", "indian":"0", "greek":"1"}
