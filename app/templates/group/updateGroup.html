POST group/updateGroup

Creates a new group or modifies an existing one.

Request Information

URI Parameters

None.

Body Parameters

User Fields

Name Description Type Additional Information
groupId The group's ID String Required
if the group does not have an Id, set to 0 to create a new group
userId The user's ID String Required if the group is being created
name The group's name String Required
users The users who belong in the group String Required
does not include the creator, comma delimited

Request Formats

application/json

{"groupId":"0", "userId":"1", "name":"foodGroup5", "users":"2, 3"}

Response Information

Resource Description

application/json

Response Formats

application/json

{
  "data": {
    "groupId": 4,
    "success": true
  }
}

or if failed

{
  "data": {
    "error": "reason for failure",
    "success": false
  }
}
